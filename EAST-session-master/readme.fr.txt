Version 1.0 de EAST qui contient :

- config_EAST

   Ce répertoire est à placer dans le même répertoire que le fichier XML 
qui va être la source de la présentation
   Il contient les fichiers CSS de EAST ainsi que les librairies 
Javascript de timesheets

- config_Jaxe

   Ce répertoire contient tous les fichiers nécessaires à l'éditeur XML Jaxe
   téléchargeable ici : http://jaxe.sourceforge.net/fr/

   Copier tous les fichiers de ce répertoire dans le répertoire "config" 
de Jaxe.

   Ce sont : la feuille de style XSLT qui transforme le XML en HTML, le 
schéma XML utilisé et un fichier pour configurer l'IHM de Jaxe

- TutorielEAST

   Ceci est un tutoriel sur EAST réalisé avec EAST ; charger le fichier 
HTML east.html dans Firefox pour le visualiser.

Si vous voulez créer une nouvelle présentation :

- lancez Jaxe ( par jaxe.bat sous Windows ou jaxe.sh sous Linux )
- choisissez "Nouveau" puis "EAST"
